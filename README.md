هذا المستودع يحتوي على الكود المصدري الجديد لموقعي الشخصي (Portfolio) كمحلل بيانات وخبير تعلم آلة. تم إعادة بناء الموقع بالكامل ليصبح **تطبيق صفحة واحدة (Single Page Application - SPA)** فائق السرعة، خالٍ تمامًا من تكرار الأكواد، وبثنائية لغة متكاملة وفورية.

---

## 🎯 الهوية والتصميم الجديد

تم تصميم الموقع ليكون بطاقة مهنية تفاعلية تعكس مهارات تحليل البيانات والذكاء الاصطناعي، باستخدام جماليات متطورة:
* **نظام تصميم هجين (Dark/Light Modes):** وضع داكن افتراضي فاخر (Deep Space Blue) مع إمكانية التبديل للوضع الفاتح بلمسة واحدة.
* **تأثيرات زجاجية (Glassmorphism):** استخدام شفافية متناغمة ومرشحات ضبابية (`backdrop-filter`) على شريط التنقل والبطاقات.
* **تفاعل البيانات الحي (Data Analyst Sandbox):** لوحة تحليلات مصغرة وتفاعلية على الصفحة الرئيسية تتيح للزوار فلترة البيانات ورؤية الرسوم البيانية تتحدث أمامهم في الوقت الفعلي.
* **معالج بدء المشاريع (Project Wizard):** معالج متعدد الخطوات يسهل على العملاء تحديد نطاق مشاريعهم وإرسالها لك مباشرة.

## 🛠️ التقنيات المعمارية الحديثة

تم بناء النظام بدون أي إطارات عمل خارجية ثقيلة (مثل React أو Vue) للحفاظ على سرعة تحميل قصوى (Zero-bundle overhead):

* **Vanilla JS ES Modules:** استخدام وحدات جافا سكريبت الأصلية (`import/export`) لتقسيم الكود البرمجي بشكل نظيف ومهني.
* **Hash-Based Router:** راوتر مخصص في المتصفح (`js/router.js`) يقرأ الروابط ديناميكيًا ويقوم بتبديل الصفحات وتحديث الـ SEO (العناوين والوصف) فوريًا دون الحاجة لخادم أو تحديث الصفحة.
* **بنية بيانات موحدة (Unified Database):** يتم تخزين جميع المقالات والمشاريع والترجمات في ملف `js/data.js` واحد ثنائي اللغة. إضافة محتوى جديد لا تتطلب تعديل أي ملف HTML.
* **تتبع ديناميكي (Dynamic Google Analytics):** إرسال أحداث تتبع الزيارات (Page Views) تلقائيًا عند التنقل بين صفحات الـ SPA لضمان دقة التحليلات.
* **توجيه ذكي 404 (Intelligent Redirect):** ملف `404.html` مخصص يقوم بتحويل الزوار ومحركات البحث القادمين من روابط قديمة (مثل `/ar/about/`) تلقائيًا إلى هيكل الـ SPA الجديد (مثل `/#/ar/about`) لمنع الروابط المكسورة والحفاظ على الأرشفة.

## 🗂️ هيكل المجلدات الجديد

```text
portfolio/
├── index.html              # الحاضن الوحيد للتطبيق (Layout, Meta)
├── 404.html                # محرك تحويل الروابط القديمة
├── style.css               # نظام التنسيق والتصميم الموحد
├── CNAME                   # النطاق المخصص لـ GitHub Pages
├── assets/                 # مجلد الصور والملفات
└── js/                     # الأكواد البرمجية للموقع
    ├── app.js              # نقطة البدء وتهيئة النظام
    ├── data.js             # قاعدة بيانات المشاريع والمقالات والترجمات
    ├── router.js           # محرك التنقل الديناميكي
    ├── components.js       # العناصر المشتركة (Navbar, Footer, Sandbox Chart)
    └── views/              # صفحات التطبيق الفرعية الديناميكية
        ├── home.js         # الرئيسية مع Dashboard مصغر
        ├── about.js        # صفحة عني والمقارنة والجدول الزمني
        ├── services.js     # صفحة الخدمات بأسلوب Bento
        ├── portfolio.js    # معرض المشاريع وتفاصيل دراسات الحالة
        ├── blog.js         # المدونة وتفاصيل المقالات
        ├── startProject.js # معالج المشاريع التفاعلي
        └── contact.js      # صفحة تواصل معي
```

## 🚀 النشر والصيانة

الموقع جاهز تمامًا للنشر المباشر على **GitHub Pages**:
* لا يتطلب أي خطوات بناء أو تجميع (No build step).
* لإضافة مشروع أو مقال جديد، قم فقط بتعديل الكود في ملف `js/data.js` وسيتم تحديث الموقع فورًا بعد الرفع للمستودع.
