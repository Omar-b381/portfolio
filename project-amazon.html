أداة استخراج بيانات المنتجات من أمازون

تطوير أداة قوية لاستخراج معلومات المنتجات (العنوان، السعر، التقييم ) من صفحات نتائج البحث في موقع أمازون.

Web Scraping Python BeautifulSoup Requests Data Mining

المهمة

بناء أداة فعالة لجمع بيانات المنتجات من أمازون وتصديرها إلى ملف Excel لتحليل المنافسين والأسعار.

الأدوات المستخدمة

Python, BeautifulSoup, Requests, Pandas

النتيجة الرئيسية

أداة قادرة على استخراج بيانات عشرات المنتجات من صفحة واحدة وحفظها في ملف Excel منظم.

🎯 الموقف (Situation )

موقع أمازون هو مصدر غني بالبيانات حول المنتجات والأسعار والتقييمات. الشركات والمحللون يحتاجون إلى هذه البيانات لتحليل السوق، مراقبة المنافسين، وتحديد استراتيجيات التسعير. جمع هذه البيانات يدوياً أمر غير عملي ويتطلب أداة آلية.

📝 المهمة (Task)

كان الهدف هو بناء أداة استخراج بيانات باستخدام Python تقوم بالآتي:

  1. إرسال طلب إلى صفحة نتائج بحث أمازون لكلمة مفتاحية معينة (مثل "laptops").
  2. تحليل محتوى HTML للصفحة المستلمة.
  3. استخراج البيانات الأساسية لكل منتج في الصفحة: اسم المنتج، السعر، وتقييم العملاء.
  4. حفظ البيانات المستخرجة في ملف Excel منظم لسهولة التحليل والمشاركة.

🛠️ الإجراء (Action)

تم استخدام مجموعة من المكتبات الفعالة في Python لإنجاز المهمة:

  • Requests: لإرسال طلبات HTTP إلى خادم أمازون والحصول على محتوى الصفحة. تم تضمين (Headers) في الطلب لمحاكاة متصفح حقيقي وتجنب الحظر.
  • BeautifulSoup: هي المكتبة الأساسية التي تم استخدامها لتحليل (Parse) محتوى HTML والبحث عن العناصر التي تحتوي على البيانات المطلوبة باستخدام محددات CSS.
    • - Pandas: لتجميع البيانات في DataFrame وتصديرها إلى ملف Excel باستخدام دالة `to_excel`.

📈 النتيجة (Result)

أثمر المشروع عن أداة بسيطة لكنها قوية جداً، وحققت النتائج التالية:

  • كفاءة في جمع البيانات: تمكين المستخدم من جمع بيانات عشرات المنتجات في ثوانٍ معدودة.
  • بيانات منظمة: إنتاج ملف Excel جاهز للاستخدام مباشرة في التحليلات أو العروض التقديمية.
    • - أساس قابل للتطوير: يمكن بسهولة تطوير هذه الأداة لتصفح صفحات متعددة أو لجمع المزيد من التفاصيل من صفحات المنتجات الفردية، مما يجعلها أساساً قوياً لمشاريع تحليل بيانات أكثر تعقيداً.
