Machine Learning Project: Breast Cancer Prediction

Building a machine learning model to classify breast tumors (malignant or benign) with high accuracy using the Wisconsin Cancer dataset.

Machine Learning Python Scikit-learn Classification EDA

Task

Develop a classification model capable of distinguishing between malignant and benign tumors with an accuracy exceeding 95% to aid in early diagnosis.

Tools Used

Python, Scikit-learn, Pandas, Seaborn, Matplotlib

Key Result

A Support Vector Classifier model was built that achieved an impressive accuracy of 98% on the test dataset.

🎯 Situation

Accurate and rapid diagnosis of breast cancer is crucial for increasing the chances of recovery. Doctors rely on numerous tests, and data modeling can be a powerful tool to automatically and accurately analyze the cellular characteristics of tumors and classify them.

📝 Task

The goal was to use the "Breast Cancer Wisconsin" dataset to build an integrated machine learning model that performs the following tasks:

  1. Exploratory Data Analysis (EDA): Understand the data characteristics, distribution, and identify the most influential factors for classification.
  2. Data Preprocessing: Process and prepare the data for model training, including handling any missing data and scaling the data.
  3. Model Building and Evaluation: Train a robust classification model and test it using precise metrics such as the Confusion Matrix and Classification Report.

🛠️ Action

The project was implemented using Python and specialized scientific libraries:

  • Data Loading: The dataset was loaded directly from the Scikit-learn library.
  • Visual Analysis: Using Seaborn and Matplotlib, visualizations (like Pairplots and Countplots) were created to understand the relationships between variables and the distribution of classes (benign/malignant).
  • Data Splitting: The data was split into Training and Test sets to ensure an objective evaluation of the model's performance.
  • Model Building: A **Support Vector Machine (SVM)** model, specifically the Support Vector Classifier, was chosen and trained due to its high capability in handling complex classification tasks.

📈 Result

The project achieved excellent results and demonstrated the power of machine learning in the medical field:

  • Superior Accuracy: The model achieved an accuracy of up to **98%** on the test data, making it a highly reliable tool for classification.
  • Clear Insights: The Classification Report showed that the model performed excellently on both Precision and Recall metrics for both classes.
  • Applicable Model: This project serves as a strong foundation that can be developed into a diagnostic support tool for doctors, accelerating the decision-making process and reducing the likelihood of human error.
