Car Data Scraping Tool from Dubizzle

Building a Web Scraper using Python and Selenium to automatically collect car ad data from the Dubizzle website.

Web Scraping Python Selenium Data Collection Automation

Task

Automate the process of collecting car data (such as price, model, mileage) from multiple pages on the Dubizzle website.

Tools Used

Python, Selenium, WebDriver Manager, Pandas

Key Result

An effective tool capable of extracting data from 50 pages and saving it into an organized, analysis-ready CSV file.

üéØ Situation

Analyzing the used car market requires large amounts of up-to-date data. Manually collecting this data from sites like Dubizzle is practically impossible, as it is time-consuming and prone to human error. The challenge was to find an automated way to collect this data efficiently.

üìù Task

The goal was to build a data scraper that performs the following tasks:

  1. Browse the Dubizzle site and navigate through pages of car ads.
  2. Extract specific information from each ad (price, car name, specifications).
  3. Aggregate all extracted data into a structured format.
  4. Save the final data into a CSV file for easy use in future analyses.

üõ†Ô∏è Action

The tool was built using Python and the following libraries:

  • Selenium: Used to control the web browser automatically, allowing the tool to interact with page elements (like clicking the "Next Page" button) that might not respond to standard requests.
  • WebDriver Manager: To facilitate the management of browser drivers (WebDriver) and avoid compatibility issues.
  • Pandas: Used to organize the extracted data into a DataFrame (a data table) and then easily export it to a CSV file.
  • Handling Challenges: The code was designed to be robust and capable of handling different page load times and avoiding related errors.

üìà Result

The project successfully developed a powerful and independent data extraction tool, achieving the following results:

  • Full Automation: A manual process that took hours was transformed into an automated one that takes minutes.
  • High-Quality Data: A clean and organized database containing hundreds of ads was collected, ready for use in data analysis or machine learning projects (such as building a car price prediction model).
  • Applicable Skill: This project demonstrates the ability to handle the challenges of scraping data from dynamic websites, a key skill for any data analyst.
